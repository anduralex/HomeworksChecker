What's this about?

This Django application is an example I created to make programming more accessible. I build this powerful yet simple tools and platforms for teachers and students... In this application, users can sign up as a student or a teacher. Teachers can create exercises and students can solve them based on their interests.

Plagiarism Checker

It's another task of this platform that can detect similar project code with unauthorized copy-paste. It was designed to use the teacher digital open database to search the similarities.




Python compiler

Allows students to compile the source code and execute it. This online compiler has the same basic functionality as a conventional Python compiler. Students can send their code to examination in a very simple and fast way.
